Amazon Prime Movies & TV Shows

This API uses a database of movies and TV shows on Amazon Prime. It works by having the user input certain values in the text boxes, so that the system can only choose movies and shows which suit that criteria. For example, if the user enters "Comedy" in the "Genre" box, then they will only get results of comedic movies and shows. They can also enter values in multiple text boxes. For example, if they enter "Comedy" in "Genre" AND "Germany" in "country", then they will get comedies made in Germany.
